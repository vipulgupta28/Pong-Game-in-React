# Pong Game in React 🎮

A classic Pong game recreated using **React.js**. This fun and interactive game demonstrates the power of React for building dynamic, real-time applications.

---

## 🚀 Features

- 🏓 Smooth gameplay with real-time ball and paddle movement
- 🎮 Keyboard controls for player paddles
- 💻 Fully responsive design for different screen sizes
- 🔄 Dynamic ball speed and bounce logic
- 🕹️ Single-player or two-player modes

---

## 🛠️ Technologies Used

- **React.js**: Frontend framework for building the game interface and logic
- **CSS**: For styling and responsiveness
- **JavaScript**: For implementing game logic and interactivity

---

## Learnings

- Understanding the integration of real-time game mechanics using React's state and effects
- Mastering the use of React hooks like useState and useEffect for game state management
- Improving CSS skills for responsive design and animations
- Enhancing JavaScript proficiency by implementing game physics (e.g., ball movement, paddle collision)
- Gaining insights into handling user input (keyboard events) for real-time interactions
